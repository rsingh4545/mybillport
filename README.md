# 📡 MyBillPort

This is the MVP version of the MyBillPort app.

## Features
- View all bills in one dashboard
- AI-based reminders for priority payments
- Upload and scan bills (OCR future)
- Credit card reminder integration
- Mobile-first design with dark mode

## 🔧 How to Run
```bash
npm install
npm run dev
```

Visit: http://localhost:3000
